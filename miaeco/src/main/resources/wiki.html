MiaEco — 参数化程序化森林

框架版本 通过规定区域与树种，自动依地形/坡度/方块/水参数选点，并异步并行地把每棵树程序化生长为形态各异、大小不一的树；可随时间推进森林演替。

快速上手

# 1. 站在一角，设置选区两个对角点
/miaeco pos1
（走到对角）/miaeco pos2

# 2. 用选区建立一片森林
/miaeco forest create 云杉林

# 3. 添加树种（使用 config.yml 的默认参数，可在 forests.yml 细调）
/miaeco species add 云杉林 spruce

# 4. 参数化选点 + 种下树苗（异步）
/miaeco plant 云杉林

# 5. 推进时间，观察长大 / 枯死 / 倒伏
/miaeco advance 云杉林 3
/miaeco advance 云杉林 12

命令一览

命令说明
/miaeco pos1 | pos2把脚下方块设为选区角点
/miaeco forest create/list/info/remove森林管理
/miaeco species add/list树种管理（参数取自 config 默认，可在 forests.yml 覆盖）
/miaeco plant <森林>异步选点 + 种植
/miaeco grow <森林>并行生成并写入待更新的树
/miaeco advance <森林> <月>推进演替
/miaeco clear <森林>清除该森林所有树方块

参数模型

选点适宜度：表面方块白名单、海拔区间、坡度上限、与水的距离（配合 water-affinity 喜湿/喜旱）。

生长形态：每个树种一个独立结构模型(TreeForm)，且每个阶段单独设计（幼树不是缩小的成树）：橡树=主干分枝圆冠→老树板根粗干；白桦=纤细白杆紧凑小冠+小侧枝；深色橡木=2x2 粗短干+宽平穹冠；丛林=裸干少量中段枝+有界伞冠+爬藤垂藤+板根，老树 2x2 巨柱；云杉=细单杆+层叠加宽的裙状叶盘成尖锥+粗壮根领，老树带露干带；金合欢=弯干上举分叉+平顶伞，老树双层伞。早期 SEED/SAPLING 皆为小芽/小苗（按树种微调样貌）。

预设体块(Stamp)：树冠裂片、板根、小侧枝、枯桩等为手工设计的体素块，程序化选位后旋转盖印，细节呈“设计感”而非噪声球。

巨大化变异：约 2.5% 的树由种子确定性判定为巨木，从幼树期即按巨木结构生长（2x2 巨干，深色橡木巨木为 3x3；巨桦只增高不增粗）。测试：/miaeco test oak plant giant。

演替：advance 按月推进年龄，活体树随年龄长大，老树随机枯死成枯立木，枯立木随机倒伏，倒木久置后回归林窗。所有随机性由树种子决定，可复现。

性能

选点、形态生成均在工作线程并行完成；方块写入分 tick 回主线程（engine.blocks-per-tick）。区域超过 50 万格会被拒绝以保护主线程。线程数见 engine.worker-threads。

路线图（框架待扩展）

  • 邻域元胞规则：竞争、林窗更新、火灾/病害传播。
  • 更真实的物理：分支承重断裂、根系随坡度、光照遮蔽影响生长势。
  • 多树种混交演替（阳性先锋 → 阴性顶极）。
  • 更丰富的方块调色板（不同 log/leaf、真菌、灌木层）。
