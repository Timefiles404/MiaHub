MiaAtlas —— 定制地图世界生成器

MiaAtlas 生成「环形分流 / 扇区轮盘」布局的定制世界：中央核心岛 → 环形海 → 12 扇区轮盘 → 四周外海。纯地形 + 纯群系：不生成任何结构建筑、树木植被、 原版装饰（唯一例外：委托规格指定的沙漠地下繁茂洞穴植被）。山体等大型地貌留白， 供后期手工塑造。

快速开始

/miaatlas create wheel                       # basic 模式，随机种子
/miaatlas create wheel mode=basic seed=123   # 指定种子
/miaatlas tp wheel                           # 传送查看
/miaatlas pregen wheel                       # 预生成全图（断点续跑，交付前必做）

别名 /atlas。权限 miaatlas.use（默认 op）。

三种细节源（mode）

模式说明要求
basic程序噪声细节，秒级建世界（默认）无
diffusionMiaEco 扩散管线出真实地貌纹理（分位归一化后按扇区调制，宏观轮盘布局不变） 服务器装有 MiaEco；首次使用自动下载模型权重（约 2GB，CPU 推理需较长时间）
import导入外部高度图作细节源（16-bit 灰度 PNG 最佳，普通图按亮度） 把 PNG 放到 plugins/MiaAtlas/import/，加 file=xx.png

提示：可先在本地用 MiaEco 的 Terra Studio 生成/挑选高度图，再以 import 模式喂给服务器—— 免去服务器上跑推理。

特制地下（三处）

① 蓝洞系统（海洋扇区，坐标见 /miaatlas info）：礁盘浅台 → 半径 52 陡壁深洞（洞底 Y-20）→ 4 条 1~3 格宽扭曲死亡竖井 → 基岩层 75×75 圆角方形暗室 （地面 Y-59、穹顶至 Y-46，基岩/深板岩地面 + 岩浆眼气泡柱）。整套系统全水密封—— 潜水挑战，无流体坍塌风险。

② ★ 深暗之域（西南外海地下，全图唯一）：中央大厅 + 4 卫星室 + 连接隧道的 密封洞窟，幽匿块/幽匿感测体/尖啸体（可召唤监守者）/催发体铺装；群系 deep_dark 仅覆盖此处。 无地表入口——需要挖掘寻找。

③ 沙漠繁茂洞穴：沙漠扇区地下 Y-28~52 为 lush_caves 群系，天然洞穴加宽加密， 苔藓地面、发光浆果藤、孢子花、黏土洼。

全局地下规则

  • 深暗之域：除 ★ 外全图禁止（群系逐格判定，不可能泄漏）。
  • 试炼大厅及一切原版结构：全图禁止（自定义生成器不产结构）。
  • 其余区域：天然洞穴（意面+奶酪+深处熔岩）正常生成；蓝洞/深暗之域周边自动禁刻保密封。
  • 矿物：原版风格分布（煤铁铜金红石青金钻，深板岩变体），ores.enabled: false 可关。

命令一览

/miaatlas create <名> [mode=basic|diffusion|import] [seed=N] [file=xx.png]
/miaatlas pregen <世界>     预生成全图（约 22 万区块；进度播报，随时 stop 暂停，游标续跑）
/miaatlas stop <世界>       暂停预生成
/miaatlas list | tp | info | remove

交付流程建议：create → tp 抽查 → pregen 跑完 → 关服 → 打包世界文件夹交付。 remove 只注销登记，不删世界文件夹。
