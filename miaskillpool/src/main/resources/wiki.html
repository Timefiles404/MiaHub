MiaSkillpool

MiaSkillpool 是依赖 MythicMobs 的技能池插件，提供技能学习、技能池 GUI、五个装备槽、槽位升级、随机抽取装配和三种释放资源模式。

前置依赖

运行前需要安装并加载 MythicMobs。通过 MiaHub 安装时可以使用：

/miah install miaskillpool --deps

玩家命令

/mias
/mias open
/mias mode <health|rage|mana>
/mias upgrade slot <1-5>

管理命令

/mias learn <玩家> <skillId>
/mias givebook <玩家> <skillId> [数量]
/mias random <玩家> roll
/mias mana addmax <玩家> <数值>
/mias reload

随机装配

/mias random <玩家> roll 会为在线玩家打开随机技能装配界面。上方两行显示 14 个随机候选技能，候选可以重复；动画结束后点击上方技能放入下方五个槽位，点击下方槽位可归还。选满五个后界面自动关闭并立即写入玩家装备槽。

释放方式

玩家将客户端“交换副手”键绑定为 Z。服务端监听副手交换事件并取消原版交换，热键栏 1-5 对应五个技能槽。

资源模式

模式说明
health消耗生命值，消耗较少，但槽位冷却更长，最低保留半颗心。
rage消耗怒气，默认 0，进入战斗后恢复。
mana消耗法力值，按配置自然缓慢恢复。
